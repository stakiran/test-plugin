# ランダム翻訳スキル（3パターン・訳文のみ）

与えられた「一言」を、ランダムに選んだ3つの自然言語へ翻訳し、**翻訳結果だけ**を表示するスキル。スキル1の改良版。

## 入力の扱い

- 対象は **一言＝一文程度**（日本語換算でおおむね100文字未満）の短いテキスト。
- 与えられたテキストが長い場合は、**最初の一文だけ**を取り出して翻訳対象にする。残りは無視してよい。
- 翻訳対象が空、または言葉として成立していない場合も、案内文は出さず何も表示しない（もしくは空行のみ）。

## 手順

1. 入力から翻訳対象の「一言」を確定する（長ければ一文だけ抜き出す）。
2. 下記「言語リスト」から**ランダムに3つ**、互いに異なる言語を選ぶ。
   - 乱数の代わりに、入力テキストの文字数などから決定的に選んでもよいが、毎回できるだけ違う言語になるよう散らすこと。
   - 3つの言語が重複しないようにする。
3. それぞれの言語へ翻訳する。
4. **翻訳結果の3行だけ**を表示する。1行につき1つの訳文。

## 出力ルール（重要）

- 案内文・見出し・言語名・原文・補足・発音・絵文字などは **一切表示しない**。
- 表示するのは翻訳後のテキストのみ。3パターンなので **3行**。
- 番号付けや記号（`-` や `1.` など）も付けない。訳文をそのまま3行並べる。

