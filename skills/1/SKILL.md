# ランダム翻訳スキル

与えられた「一言」を、ランダムに選んだ自然言語へ翻訳して表示するスキル。

## 入力の扱い

- 対象は **一言＝一文程度**（日本語換算でおおむね100文字未満）の短いテキスト。
- 与えられたテキストが長い場合は、**最初の一文だけ**を取り出して翻訳対象にする。残りは無視してよい。
- 翻訳対象が空、または言葉として成立していない場合は、その旨を伝えて終了する。

## 手順

1. 入力から翻訳対象の「一言」を確定する（長ければ一文だけ抜き出す）。
2. 下記「言語リスト」から**ランダムに1つ**言語を選ぶ。
   - 乱数の代わりに、入力テキストの文字数などから決定的に選んでもよいが、毎回できるだけ違う言語になるよう散らすこと。
   - 直近で選んだ言語となるべく重複させない。
3. 選んだ言語へ翻訳する。
4. 次の形式で結果を表示する。

   ```
   🌐 選ばれた言語: <言語名（日本語）> / <現地表記や英語名>
   原文: <入力の一言>
   訳: <翻訳結果>
   （補足: <発音のカナ・直訳・豆知識など。任意>）
   ```

## 例

入力: `おはよう`

```

（補足: 「ヒュオメンタ」。Hyvää huomenta と言うとより丁寧）
```
